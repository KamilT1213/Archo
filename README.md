# Archo Demo

Demo of digging and extraction game set in space, Hyper casual, rpg-like.

---

## Requirements

- Visual Studio or VS Code
- CMake versions 3.10 - 3.29
- System that supports OpenGL version 4.6 or better
- Mouse for input

---
